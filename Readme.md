# Tetris-rb

Tetris game playable in console. Has support for ANSI colors.

## Run the game

```bash
git clone https://github.com/dominikb/tetris-rb
cd tetris-rb
ruby tetris.rb
```

## Supported Controls

| Key   | Action     | 
|-------|------------|
| ←     | Move Left  |
| →     | Move Right |
| ↑     | Rotate     |
| ↓     | Move Down  |
| q     | Quit       |
| p     | Pause      |
| r     | Restart    |
| Space | Drop       |
